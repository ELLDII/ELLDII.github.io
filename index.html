Технические проекты (Инфраструктура и разработка)

1. Организация low-latency сети для управления платформой Unitree Go2

Стек: TCP/IP, DHCP, Wi-Fi 802.11, LAN маршрутизация, Low-latency

Задача: Обеспечить бесперебойную передачу потоковых команд управления между сервером (ПК) и клиентом (роботом).

  • Настроил локальную сетевую инфраструктуру через Wi-Fi роутер.
  • Провел диагностику сети, выявил потерю пакетов и высокий пинг в беспроводном канале.
  • Спроектировал и перевел систему на проводное соединение (RJ45), обеспечив минимальную задержку сети и максимальную стабильность для передачи данных реального времени.

2. Разработка конфигуратора и настройка связи для БПЛА

Стек: Радиоканалы, Linux/RTOS, GUI/CLI утилиты, Удаленное администрирование

Задача: Настройка беспроводных мостов передачи данных и создание панели администрирования.

  • Настроил и откалибровал радиоканалы для передачи телеметрии и удаленного мониторинга оборудования.
  • Написал ПО для полетного контроллера.
  • Разработал конфигуратор (утилиту) для удаленного администрирования, настройки параметров системы и диагностики.

3. Интеграция IoT-робота (ESP32) с веб-сервисами и AI

Стек: HTTP/HTTPS, REST API, JSON, клиент-серверная архитектура, DNS

Задача: Организовать сетевой стек на микроконтроллере для общения с внешними веб-серверами.

  • Настроил Wi-Fi клиент и безопасное подключение (SSL/HTTPS) к сети интернет.
  • Реализовал отправку и прием REST API запросов (GET/POST) к облачным моделям.
  • Настроил парсинг JSON-ответов для взаимодействия устройства с пользователем в реальном времени.

4. Разработка и деплой модуля для электронного дневника

Стек: Веб-администрирование, Базы данных, Деплой, Интеграция

Задача: Внедрение нового веб-модуля в существующую образовательную платформу.

  • Разработал логику работы модуля и его связь с базой данных.
  • Выполнил деплой (развертывание) на сервере и интеграцию в основную архитектуру сайта.
  • Обеспечил корректную работу модуля при пользовательских нагрузках.
